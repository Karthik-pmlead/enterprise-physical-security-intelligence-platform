# Platform Case Studies

The Enterprise Physical Security Intelligence Platform is designed as a reusable decision platform.

Although inspired by enterprise access control, the same architecture can support multiple operational domains by reusing the same core capabilities.

## Included Case Studies

- Enterprise Access Control
- Fleet Intelligence
- Smart Buildings
- Industrial IoT

Each case study demonstrates how identity, telemetry, risk, policy, and real-time decisioning can be adapted to different business problems while leveraging the same platform architecture.
